# archivesspace-api-training-codespaces

A codespace initially created for session 3 of the April 2026 ArchivesSpace API training.

Topics covered include:

- Very high-level overview of creating a devcontainer
- Basic credential management (and light git best practices)
- ArchivesSnake in a Python REPL
- Using ArchivesSnake within a Python script
- Basic GET, POST, and DELETE requests via Python scripts
